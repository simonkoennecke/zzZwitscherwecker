Datenschutz

Wir möchten Sie drauf hinweisen, dass die App mit einem Server Kommuniziert. Es werden keine Person bezogenen Daten erhoben. Nach einem Quiz werden an www.farbtrommel.de folgende Werte übermittelt:

  • Weckerzeit
  • Dauer
  • Anzahl der Versuche
  • Auswahlmöglichkeiten
  • Richtige Antwort
  • Anonymisierte UUID

Alle erhobene Daten werden unter www.farbtrommel.de veröffentlicht.
